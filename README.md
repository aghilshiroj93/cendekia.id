# CENDEKIA.ID - Professional Company Profile & Digital Solutions

Website Profil Perusahaan modern untuk **CENDEKIA.ID**, dirancang dengan fokus pada estetika premium, performa tinggi, dan kemudahan pengelolaan konten melalui Dashboard Admin yang fungsional.

## 🚀 Fitur Unggulan

- **Premium Landing Page**: Desain bersih, modern, dan profesional dengan skema warna Amber & White.
- **Advanced Animations**: Menggunakan sistem *Scroll-Reveal* dan *Staggered Animations* yang halus untuk pengalaman pengguna yang mewah.
- **Full-Featured Admin Dashboard**:
  - **Kelola Layanan**: Tambah/Edit/Hapus layanan jasa yang ditawarkan.
  - **Kelola Portofolio**: Dokumentasikan proyek sukses dengan detail modal yang interaktif.
  - **Kelola Testimoni**: Kelola bukti kepuasan pelanggan (Social Proof).
  - **Kelola Sosmed**: Update tautan media sosial secara dinamis.
  - **Kelola Info Bisnis**: Ubah Alamat, WA, Email, dan Jam Operasional secara langsung ke Footer.
- **Security**: 
  - Proteksi halaman Admin dengan sistem Login.
  - **Auto-Logout**: Keamanan otomatis yang mengeluarkan sesi admin jika tidak ada aktivitas selama 5 menit.
- **Responsive Design**: Optimal di semua perangkat (Desktop, Tablet, dan Smartphone).
- **Floating WhatsApp**: Tombol kontak cepat yang terintegrasi langsung dengan nomor bisnis.

## 🛠️ Teknologi yang Digunakan

- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Styling**: Vanilla CSS & CSS Modules (Modular & Performa Tinggi)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: React Hooks (useState, useEffect, useRef)
- **Data Persistence**: LocalStorage API (dengan sistem sinkronisasi data)
- **Animations**: Custom CSS Keyframes & Intersection Observer API

## 📦 Instalasi & Penggunaan Lokal

1. **Clone Repositori**:
   ```bash
   git clone https://github.com/aghilshiroj93/cendekia.id.git
   ```

2. **Masuk ke Direktori**:
   ```bash
   cd cendekia.id
   ```

3. **Instal Dependensi**:
   ```bash
   npm install
   ```

4. **Jalankan Mode Pengembangan**:
   ```bash
   npm run dev
   ```
   Buka [http://localhost:3000](http://localhost:3000) di browser Anda.
